# “少年时”——学习任务管理系统

## 在线地址

[https://youky.top/youth-values/](https://youky.top/youth-values/)

## 本地运行

```shell
git clone https://github.com/Youky1/youth-values.git
cd youth-values
npm run dev
```

## 技术栈

- 前端：`React` + `Redux` + `react-router` + `TypeScript`
- 后端：`Koa` + `Sequelize`
- 数据库：`MySQL`
